## 简介

`ls`，是类Unix系统中的指令，**用于列出文件和目录**，是英文`list`的缩写。此命令最早产生于Multics系统，并被后来的Unix和Linux系统所继承。

## 使用

列出详细信息：
```shell
ls -l
```
有些Linux系统还提供了 `ll` 命令，ll 是 `ls -l` 的别名。

查看目录下所有内容，包括隐藏文件：
```shell
ls -a [path]
```

查看目录下所有内容，并列出详细信息：
```shell
ls -al
```

查看inode号：
```shell
ls -i
```

区分文件：
```shell
ls -F   # 主要区分目录文件、可执行文件、链接文件，并且在末尾加上 / 、*、@符号标识
```

以人类易读的方式显示文件大小：
```shell
ls -h
```
