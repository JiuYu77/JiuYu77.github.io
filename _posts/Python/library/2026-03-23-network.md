## pywifi

这是一个WIFI库。pywifi模块基于comtypes模块。
pywifi是在python中一个用于操作无线接口的模块，可以跨平台使用，Windows和Linux都支持。

pip安装命令：
```bash
pip install pywifi
此库基于comtypes，若没有安装，则需要安装comtypes：
pip install comtypes
```
导包示例：
```python
import pywifi
from pywifi import const  // 需要用时导入
```
